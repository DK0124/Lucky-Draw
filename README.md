# 🎊 通用抽獎系統

適用於任何公司活動的通用抽獎系統，具有多種刺激視覺與音效效果的 Web 抽獎器。

## ✨ 核心特色

### 🎯 完全可自訂
- **活動資訊**：自訂活動名稱與副標題
- **獎品管理**：自由新增、編輯、刪除、排序獎品
- **參與人數**：彈性設定參加人數（號碼範圍 1~N）
- **資料持久化**：所有設定自動儲存至瀏覽器 LocalStorage
- **匯入/匯出**：可將設定匯出為 JSON 檔案，並從檔案匯入

### 🎨 視覺與動畫
- **五種動畫模式**：福袋球池、金運輪盤、福星雷達、紅包拉霸、抽籤桌面
- **春節主題**：紅金配色，喜慶氛圍
- **福袋球池**：碰撞物理效果，最後一球中獎
- **全螢幕模式**：適合投影播放

### 🎵 音效系統
- **內建音效**：各動畫專屬音效、大獎歡呼聲
- **拉霸機**：經典吃角子老虎音效
- **無需外部檔案**：使用 WebAudio API 合成音效
- **可開關**：隨時切換音效開關

### 📋 獎項管理
- **完整獎品資訊**：名稱、數量、描述
- **即時狀態顯示**：剩餘數量、抽出進度
- **靈活操作**：編輯、刪除、上下移動排序
- **加碼驚喜**：臨時追加獎品
- **重抽功能**：允許重抽上一位得獎者

### 📊 得獎紀錄
- **舞台右側**：即時顯示得獎名單
- **管理面板**：完整歷史紀錄含時間戳記
- **清楚呈現**：號碼、獎品名稱、抽出順序

## 🚀 使用方式

### 快速開始
1. 直接開啟 `index.html`（無需安裝、無需伺服器）
2. 首次使用會自動開啟管理面板
3. 自訂活動名稱、獎品清單、參加人數
4. 按空白鍵或點擊「開始抽獎」進行抽獎

### 管理面板操作
按 `A` 鍵開啟/關閉管理面板，分為三大區塊：

#### 左欄：活動設定
- 設定活動名稱與副標題
- 設定參加人數（套用會重置抽獎進度）
- 選擇動畫模式（自動輪替或指定）
- 開關音效

#### 中欄：獎項管理
- **新增獎項**：填寫名稱、數量、說明後點擊「新增獎項」
- **編輯獎項**：點擊「編輯」按鈕修改獎品資訊
- **刪除獎項**：點擊「刪除」按鈕移除獎品（需確認）
- **調整順序**：使用「↑」「↓」按鈕調整抽獎順序
- **清空獎項**：一鍵清除所有獎品（需確認）
- **加碼驚喜**：臨時追加一個「加碼驚喜」獎品
- **重抽上一位**：將上一位得獎者號碼放回抽獎池

#### 右欄：紀錄與資料
- 查看已抽出紀錄（含時間戳記）
- 匯出設定為 JSON 檔案
- 從 JSON 檔案匯入設定
- 重置所有資料（清除 LocalStorage）

## ⌨️ 快捷鍵

- `Space`（空白鍵） - 開始抽獎
- `A` - 開啟/關閉管理面板
- `Esc` - 關閉管理面板
- 全螢幕按鈕 - 切換全螢幕模式

## 💾 資料持久化

所有設定會自動儲存至瀏覽器的 LocalStorage，包括：
- 活動名稱與副標題
- 獎品清單（名稱、數量、說明、順序）
- 參與人數設定
- 已抽出紀錄

頁面重新整理後，所有資料都會保留。若要清除資料，可使用管理面板的「重置所有資料」功能。

## 📦 匯入/匯出

### 匯出設定
點擊「匯出設定 JSON」按鈕，系統會下載包含所有設定的 JSON 檔案，檔名格式：`lottery-config-YYYYMMDD-HHMMSS.json`

### 匯入設定
點擊「匯入設定 JSON」按鈕，選擇先前匯出的 JSON 檔案，系統會自動載入所有設定。

### JSON 格式範例
```json
{
  "version": "1.0",
  "eventName": "2026 尾牙抽獎",
  "eventSubtitle": "春節特別活動",
  "participantsN": 50,
  "prizes": [
    {
      "id": "p1",
      "name": "精美文具組",
      "description": "包含筆記本、鋼筆等",
      "total": 5,
      "remaining": 5,
      "order": 1,
      "isSurprise": false
    }
  ]
}
```

## 🛠️ 技術特色

- **單一檔案**：所有功能整合在一個 HTML 檔案中
- **無需後端**：純前端實作，無需伺服器
- **零相依性**：不使用任何外部函式庫或框架
- **離線可用**：無需網路連線即可使用
- **響應式設計**：適應各種螢幕尺寸
- **現代瀏覽器**：支援 Chrome、Firefox、Safari、Edge
