# 🎊 尾牙抽獎系統

公司尾牙抽獎專用系統，具有多種刺激視覺與音效效果的 Web 抽獎器。

## 功能特色

- 可自訂參加人數（號碼範圍 1~N）
- 五種動畫模式：福袋球池、金運輪盤、福星雷達、紅包拉霸、抽籤桌面
- 豐富音效：各動畫專屬音效、大獎歡呼聲
- 福袋球池：碰撞物理效果，最後一球中獎
- 拉霸機：經典吃角子老虎音效
- 舞台右側即時顯示得獎名單
- 支援加碼驚喜與重抽功能
- 全螢幕模式適合投影播放

## 使用方式

1. 直接開啟 `index.html`
2. 按 `A` 鍵開啟管理面板設定參加人數
3. 按空白鍵或點擊「開始抽獎」進行抽獎
4. 得獎名單會即時顯示在右側

## 快捷鍵

- `Space` - 開始抽獎
- `A` - 開啟/關閉管理面板
- 全螢幕按鈕 - 切換全螢幕模式
